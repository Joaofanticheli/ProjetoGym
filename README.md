ProjetoGym

Projeto pessoal desenvolvido após 1 mês de estudo autodidata.
*Tecnologias*
HTML
CSS
JavaScript

*Objetivo*
Desenvolver um site do zero para consolidar fundamentos de HTML, CSS e JavaScript, com foco em preparação para o mercado de trabalho.

*Descrição*
Site com temática fitness, voltado a incentivar hábitos saudáveis e oferecer suporte ao usuário por meio de conteúdos, treinos e dietas.

*Aprendizado autodidata*
Estudo baseado em vídeos no YouTube e uso de IA apenas para esclarecimento conceitual, sem geração direta de código, priorizando o aprendizado do processo de escrita e lógica.
